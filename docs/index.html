Remote AgentsUzak Ajanlar

Connect From Anywhere With a Single Command Tek Bir Komutla Her Yerden Bağlanın

Run agents on any machine with the aidesk-agent npm package. Agents poll the AIDesk server over HTTP, making it work through firewalls and NAT. aidesk-agent npm paketi ile herhangi bir makinede ajan çalıştırın. Ajanlar AIDesk sunucusunu HTTP üzerinden yoklar, güvenlik duvarları ve NAT arkasında çalışır.

  • Zero configuration - just set token and server URL
  • Sıfır yapılandırma - sadece token ve sunucu URL'si ayarlayın
  • Firewall-friendly HTTP polling architecture
  • Güvenlik duvarı dostu HTTP yoklama mimarisi
  • 60-second heartbeat with auto-offline detection
  • 60 saniye kalp atışı ile otomatik çevrimdışı tespiti
  • Secure token-based authentication
  • Güvenli token tabanlı kimlik doğrulama
Terminal
# Install and run the remote agentUzak ajanı yükleyin ve çalıştırın

$ AIDESK_TOKEN=your-agent-token \
  npx aidesk-agent \
    --server https://your-server:3939

# Or install globallyVeya global olarak yükleyin
$ npm install -g aidesk-agent
$ aidesk-agent --server https://your-server:3939
